# How this wiki works

Peptidia is built around one principle: **trust through transparency**.

## Evidence tiers

Every claim is tagged by the strength of evidence behind it:

| Tier            | Meaning                                                        |
| --------------- | ------------------------------------------------------------- |
| 🟢 Clinical     | Supported by human clinical trials.                           |
| 🟡 Preclinical  | Animal or in-vitro studies only.                              |
| 🔵 Mechanistic  | Theoretical, based on known biology.                          |
| ⚪ Anecdotal    | Community and self-reports. Useful signal, not evidence.      |

## Contributing

Articles are MDX files under `apps/docs/docs/`. Each peptide article follows a shared
template (overview, mechanism, evidence by goal, safety, references). Edit links on each
page point back to the source file.
