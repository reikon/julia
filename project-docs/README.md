# project-docs

Internal planning & strategy documents that **feed development** of this project.
These are **not** published to the wiki — they are inputs Claude Code and the team use to
build and evolve `julia`.

> Distinct from `apps/docs` (the Docusaurus wiki) and the `/doc` route (`:3000/doc`, the
> public wiki). Nothing here is shown to end users.

| File             | What it is                                                        | Status      |
| ---------------- | ---------------------------------------------------------------- | ----------- |

| `torefine.md`    | Raw ChatGPT strategy convo — primary source material for the PRD. | ✅ dropped   |
| `PRD.md`         | Product Requirements Doc — to be crafted from BMC + torefine.    | ⏳ to write  |

## Workflow

1. Drop raw inputs here (BMC, the ChatGPT convo, notes).
2. Synthesize them into `PRD.md`.
3. Build from the PRD, continuing from a fresh context.
